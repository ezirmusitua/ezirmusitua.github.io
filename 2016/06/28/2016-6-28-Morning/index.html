2016-6-28 Morning

2016-06-28


Boston 的天气真是说变就变，刚才出去还是大晴天，收拾一会再看就已经是下过雨了 … …
早上没去跑步，主要原因就是懒!昨天可能太累了，早上闹钟没把我叫醒，睡到六点半才起来，真实尴尬。
正好昨天看的胸肌锻炼，果断就用那个代替跑不了，港真，那个教程真的是家庭版嘛，真的是初级嘛，我觉得我身体素质很好的嘞，尼玛我都不能完整地做一套下来 … …
昨天比较蛋疼，快走的时候测出来一堆 bug，害我改到 8 点多，累屎老子了，有一个 bug 是因为接口没对上的原因，那个 bug 其实不难，找了半天才找到的原因就是没有找到正确的数据流向，其实找的过程应该是:

  1. 查看 db 中有没有 – 是不是显示问题
  2. 查看 network 发送的请求中有没有 – 是不是前端传入的问题
  3. 查看传入参数的来源 – 回溯
  4. 从来源的最开始找问题 – 二分查找

PS. 不知道如何使用 mdChip 也是为什么我没有找到正确的数据流的原因之一
做 FreeCodeCamp 卡住了，难道真的要做一辈子前端渣，我不服，感觉目前很大的原因在于我 layout 知识的不足，打算先补一补 layout 部分的知识再说
想试着做生煎，最近准备准备材料 ~
