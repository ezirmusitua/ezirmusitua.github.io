昨天学到一点:
Lodash 的某些方法处理的对象一定要是 JSON 格式的，不然可能会用不了，比如: 

User.findById(userId).exec().then(function(user) {
    return {
        user: _.omit(user.toJSON(), 'username')
    }
})

上面代码的含义就是忽略从 mongo 得到的 user 的 username 属性，但是从 mongo 拿回来的是对象实际上是 document 对象，因此必须使用 toJSON 转化为 json 格式，否则无法利用 omit。
